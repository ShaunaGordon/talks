## Forks vs Branches

{{% columns %}}
{{% column %}}
Branches
- Dependent on repo
- Inextricably linked
- Closely related to repo
- Intended to be merged back into main branch (usually)
{{% /column %}}

{{% column %}}
Forks
- Based off a parent repo
- Separate/independent
- Not necessarily related after forking
- Merging back to parent optional

{{% /column %}}
{{% /columns %}}
